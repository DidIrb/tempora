# Welcome to Tempora

Tempora is a CLI scaffolding tool that lets you instantly bootstrap any project from a curated vault of community-driven templates.

Run one command, get a fully configured project, start building immediately.

```bash
npm install -g @tempora/cli
tempora init
```

## What you will find here

- [Getting Started](/getting-started) — install the CLI and scaffold your first project
- [CLI Reference](/cli) — all commands and flags explained
- [Templates](/templates) — browse available templates by language, category and library
- [Add a Template](/add-template) — contribute your own template to the vault
- [Changelog](/changelog) — version history

## Requirements

- **Node.js 18+**
- **git**
- **npm**

# Changelog

## 0.1.0

Initial release.

- `tempora init` with direct and guided mode
- Registry bundled with CLI — no remote API calls to browse templates
- Anti-overwrite guard on non-empty directories
- Dev mode copies from local templates folder
- Production mode fetches from GitHub
- Nextra docs site with auto-sync from template READMEs

# Changelog

## 0.1.0

Initial release.

- `tempora init` with direct and guided mode
- Guided flow — language → category → library → template
- Registry bundled with CLI, no remote API calls to browse
- Git sparse checkout for template downloading
- Anti-overwrite guard on non-empty directories
- `nextSteps` field in `tempora.json` for per-template post-install instructions
- Nextra v3 docs site with auto-sync from template READMEs

# Templates

Tempora provides a growing vault of curated project templates across languages and frameworks. Each template is a complete, opinionated starting point — install dependencies and start building immediately.

Pick a template using the CLI:

```bash
# guided mode — walks you through language and category
tempora init

# direct — if you already know the template name
tempora init nextjs-tailwind my-app
```

Browse templates by language in the sidebar. Each page includes the full stack, versions, and what to run after scaffolding.

> Don't see what you need? [Contribute a template](/add-template).

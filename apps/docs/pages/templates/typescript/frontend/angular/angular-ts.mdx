# Angular TypeScript

A standard Angular application scaffolded with the Angular CLI and TypeScript.

## Quick Start

```bash
npx tempora-cli init angular-ts my-app
```

## Stack

- Angular 18
- TypeScript 5
- Angular CLI

## Getting Started

```bash
pnpm install
pnpm start
```

Open http://localhost:4200

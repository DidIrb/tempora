# Next.js TypeScript

A Next.js app with the App Router and TypeScript, ready for full-stack development.

## Quick Start

```bash
npx tempora-cli init next-ts my-app
```

## Stack

- Next.js 14
- React 18
- TypeScript 5

## Getting Started

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000

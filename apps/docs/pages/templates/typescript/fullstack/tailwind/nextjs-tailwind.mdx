# Next.js + Tailwind

Next.js 14 app router with Tailwind CSS and Prettier preconfigured.

## Stack

- Next.js 14 (app router)
- Tailwind CSS
- TypeScript
- Prettier

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

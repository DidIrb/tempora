# Getting Started

<Steps>

### Install the CLI

```bash
npm install -g @tempora/cli
```

Verify it works:

```bash
tempora --version
```

### Browse available templates

```bash
# guided mode — walks you through language, category, then template
tempora init

# or go direct if you know the template name
tempora init nextjs-tailwind
```

### Scaffold your project

```bash
# into a new folder
tempora init nextjs-tailwind my-app

# into the current folder
tempora init nextjs-tailwind .
```

<Callout type="warning">
  Tempora will warn you if the target directory is not empty and ask for confirmation before proceeding.
</Callout>

### Follow the next steps

After scaffolding, Tempora prints exactly what to run:

```bash
cd my-app
pnpm install
pnpm dev
```

</Steps>

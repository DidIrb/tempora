# Getting Started

## Requirements

Before using Tempora make sure you have the following installed:

- **Node.js 18+** — [nodejs.org](https://nodejs.org)
- **git** — [git-scm.com](https://git-scm.com)
- **npm** — comes with Node.js

<Steps>

### Install the CLI

```bash
npm install -g @tempora/cli
```

Verify it works:

```bash
tempora --version
```

### Scaffold your project

```bash
# guided mode
tempora init

# direct — into a new folder
tempora init nextjs-tailwind my-app

# direct — into current folder
tempora init nextjs-tailwind .
```

<Callout type="warning">
  Tempora will warn you if the target directory is not empty and ask for confirmation before proceeding.
</Callout>

### Follow the next steps

After scaffolding Tempora prints exactly what to run next based on the template itself.

</Steps>

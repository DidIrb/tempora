# CLI Reference

## `tempora init`

Scaffold a project from a template.

```bash
tempora init [template] [directory]
```

| Argument | Description |
| --- | --- |
| `template` | Template ID (e.g. `nextjs-tailwind`). Optional — omit for guided mode. |
| `directory` | Target folder name. Use `.` for current directory. Optional — prompts if omitted. |

**Examples**

```bash
# guided mode
tempora init

# direct — new folder
tempora init nextjs-tailwind my-app

# direct — current folder
tempora init nextjs-tailwind .
```

---

## `tempora info <template>`

Show details about a specific template.

```bash
tempora info nextjs-tailwind
```

---

## `tempora --version`

Show the installed CLI version.

```bash
tempora --version
tempora -v
```

---

## `tempora --help`

Show all available commands and usage.

```bash
tempora --help
```

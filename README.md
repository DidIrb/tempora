# Tempora

A fast, language-agnostic CLI that bootstraps your project from a curated vault of community templates — no setup, no guesswork.

## Install

```bash
npm install -g tempora-cli
```

## Usage

**Guided mode:**
```bash
tempora init
```

**Direct mode:**
```bash
tempora init next-tailwind my-app
```

## Requirements

- Node.js 18+
- git
- npm

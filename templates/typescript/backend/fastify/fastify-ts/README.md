# Fastify TypeScript

A high-performance Fastify REST API server built with TypeScript.

## Quick Start

```bash
npx tempora-cli init fastify-ts my-app
```

## Stack

- Fastify 4
- TypeScript 5
- Node.js 20

## Getting Started

```bash
pnpm install
pnpm dev
```

Server running at http://localhost:3000

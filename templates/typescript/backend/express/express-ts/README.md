# Express TypeScript

A minimal Express.js REST API server built with TypeScript.

## Quick Start

```bash
npx tempora-cli init express-ts my-app
```

## Stack

- Express 4
- TypeScript 5
- Node.js 20

## Getting Started

```bash
pnpm install
pnpm dev
```

Server running at http://localhost:3000

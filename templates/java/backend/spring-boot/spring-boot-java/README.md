# Spring Boot Java

A standard Spring Boot REST API with Java, Spring Security, and Maven.

## Quick Start

```bash
npx tempora-cli init spring-boot-java my-app
```

## Stack

- Spring Boot 3
- Java 21
- Maven

## Getting Started

```bash
mvn install
mvn spring-boot:run
```

Server running at http://localhost:8080
